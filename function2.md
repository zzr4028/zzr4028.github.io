# 二次函数
## 概念
**定义1** 一般地，形如*y=ax²+bx+c*(a,b,c是常数且a≠0)的函数，叫做二次函数。
其中x是自变量，a,b,c分别是二次函数表达式的二次项系数，一次项系数和常数项。
> 注意：和一元二次方程类似，二次项系数a≠0，而b、c可以为零。二次函数自变量的取值范围是全体实数。

## 图象
二次函数*y=ax²+bx+c*是一个抛物状的图象，因此也叫二次函数为抛物线。

通过作图可知：二次函数的图象是轴对称图形，其对称轴是一条平行于y轴的直线。
对称轴与二次函数的交点叫做二次函数的顶点，二次函数在顶点处取得最值。
当a>0时，二次函数在顶点处取最小值；
当a<0时，二次函数在顶点处取最大值。

## 平移
平移口诀：左加右减（横坐标x），上加下减（纵坐标y）。
1. 二次函数*y=ax²+k*可以由二次函数*y=ax²向上*平移k个单位得到；
2. 二次函数*y=a(x-h)²*可以由二次函数*y=ax²*向右平移h个单位得到；
3. 二次函数*y=a(x-h)²+k*可以由二次函数*y=ax²*向右平移h个单位，再向上平移k个单位得到；

二次函数*y=ax²+bx+c*经过配方可以写成*y=a(x+b/2a)²+(4ac-b²)/4a*，这叫顶点式。

## 图象与系数的关系
1. 抛物线开口由a定，上正下负，|a|越大，开口越小；
2. 对称轴x=-b/2a的位置由a和b定，左同右异（符号），b为0时是y轴；
3. 抛物线与y轴的交点由c定，上正下负，c为0时过原点；
4. Δ=b²-4ac决定抛物线与x轴交点的个数。

## 图象中的信息
1. 根据开口方向判断a的正负性；
2. 根据对称轴在y轴的左右情况判断a、b的正负性；
3. 根据与y轴的交点判断c的正负性；
4. 根据与x轴的交点个数判断Δ=b²-4ac的正负性；
5. 根据经过点的坐标得到关于a,b,c的方程(组)；
6. 根据顶点判断(4ac-b²)/4a的大小。

> y1 = ax1²+bx1+c
> y2 = ax2²+bx2+c
> y3 = ax3²+bx3+c
